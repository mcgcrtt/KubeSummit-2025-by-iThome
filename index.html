🚀 Spring Native Workshop

學習如何建置和部署高效能的 Spring Native 應用程式

關於此 Workshop

Spring Native 讓您能夠將 Spring 應用程式編譯成原生執行檔，顯著提升啟動速度並減少記憶體使用量。 透過本實驗，您將建置一個圖片分析應用程式，整合 Google Cloud Vision API、Cloud Storage 和 Firestore， 並比較 JIT 與 Native 版本在 Google Cloud Run 上的效能差異。

實驗內容

Spring Native Workshop - Image Analysis Application

⏱️ 時間：約 50 分鐘 📊 難度：中級

建置一個完整的圖片分析應用程式，學習如何使用 Maven 建置 JIT 和 Native 版本， 整合 Google Cloud Vision API 進行圖片辨識，使用 Eventarc 建立事件驅動架構， 並將應用程式部署到 Cloud Run。透過實際案例比較兩種版本在啟動時間、記憶體使用和映像檔大小的差異。

Spring Boot GraalVM Maven Cloud Run Vision API Firestore Eventarc
